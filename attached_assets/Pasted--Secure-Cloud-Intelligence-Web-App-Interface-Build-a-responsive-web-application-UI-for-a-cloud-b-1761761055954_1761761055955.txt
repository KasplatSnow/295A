“Secure Cloud Intelligence — Web App Interface”

Build a responsive web application UI for a cloud-based security system called “Secure Cloud Intelligence.” The system provides real-time anomaly detection from live video streams. The UI should look clean, modern, and professional with a light color palette (white background, blue and gray accents). Typography should be sans-serif (Roboto or Inter). Use subtle shadows, rounded corners, and clear icons. The layout should feel like a real product dashboard, not a marketing site.

Pages to Generate

Title: “Secure Cloud Intelligence”

Subtitle: “Smart, Context-Aware Home Security”

Centered login card with:

Email field

Password field

General Design Guidelines

Theme: Light mode

Accent color: Blue (#2563EB)

Secondary color: Gray (#6B7280)

Highlight color: Orange/Red for critical alerts

Icons: Use simple vector icons (camera, chart, alert, settings, profile)

Spacing: Ample white space; padding 20–32px between sections

Charts: Simple, readable — line/bar/pie with clear legends

Typography:

Title font size: 22–24px bold

Body: 14–16px regular

Page Flow
Login → Dashboard → Incident Details → Reports → Cameras → Settings

Output Expectation

The result should be a responsive, high-fidelity web UI design mockup (or webpage layout) that looks like a real working cloud dashboard for security monitoring — not a static poster. Each page should be generated in a consistent color scheme and component style.
